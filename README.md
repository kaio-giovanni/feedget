# Feedget

Project created on NLW Return

#### Project Setup

1. Execute the commands bellow:

```bash
git clone https://github.com/kaio-giovanni/feedget.git
cd feedget
npm install
```

#### Run project

1. Execute the commands bellow:

```bash
npm run dev
```
